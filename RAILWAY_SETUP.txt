RAILWAY PROXY SETUP GUIDE
=========================

FILES TO UPLOAD:
1. index.js (the Node.js server code)
2. package.json (dependencies)

STEPS:
1. Railway → New Project → Empty Project
2. Click project → + New → Empty Service
3. Upload index.js and package.json
4. Railway auto-detects Node.js and deploys
5. Settings → Networking → Generate Domain
6. Copy your new URL (like: https://your-proxy.up.railway.app)
